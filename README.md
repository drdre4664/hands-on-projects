# Hands-On Projects

A portfolio of end-to-end cloud and infrastructure projects demonstrating practical skills in
containerization, cloud infrastructure, CI/CD automation, and configuration management.
Each project reflects real-world engineering workflows — from infrastructure provisioning
through to production deployment.

## Skills Demonstrated

- **Containerization:** Docker, Dockerfile authoring, multi-stage builds, Docker Compose, named volumes, container networking
- **Cloud:** Azure VM provisioning, Network Security Groups, public IP management, Cloud-Init bootstrap scripts
- **Infrastructure as Code:** Terraform — provisioning Azure VMs, MySQL, and networking resources
- **Configuration Management:** Ansible — playbooks, inventory management, group_vars, PM2-managed deployments
- **CI/CD:** Azure DevOps YAML pipelines, GitHub-triggered builds, secure secret management
- **Networking:** Docker bridge networks, container DNS resolution, port mapping, Nginx reverse proxy
- **App Stacks:** Next.js, Node.js/Express, MySQL/Sequelize, JWT authentication, React + Nginx
