## Core Skills Demonstrated

**Cloud Platforms**
AWS (VPC, EC2, RDS Multi-AZ, ALB, Auto Scaling, Route 53, CloudWatch) · Azure (VMs, NSG, Load Balancer, Azure DevOps, MySQL Flexible Server)

**Infrastructure as Code**
Terraform (modular architecture, remote state S3 backend, multi-environment workspaces)

**Configuration Management**
Ansible (role-based deployments, Jinja2 templates, handlers, idempotency verification)

**Containers & Orchestration**
Docker (multi-stage builds, Compose, Nginx reverse proxy, private registry)

**CI/CD Pipelines**
Azure DevOps (YAML pipelines, self-hosted agents, two-repo model: infra + app)

**Architecture Patterns**
3-tier HA design · Multi-AZ failover · Public + Internal load balancers · Security Group least-privilege · Auto Scaling with health checks
